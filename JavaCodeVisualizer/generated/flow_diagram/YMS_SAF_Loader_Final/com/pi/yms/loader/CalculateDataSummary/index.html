com.pi.yms.loader.CalculateDataSummary

This class is used to calculate the statistics based on the provided test data. BigDecimal API is used extensively in this class to calculate the values, since there are issues with primitive double values. For Ex: 0.3 * 3.0 = 0.8999999999999999 using primitive doubles

No. of Methods With Description Without Description
9 9 0
Method Description
calculateDataSummary This method takes care of calculating the statistics test with comma separated Serial, Wafer, Bin Pass Fail and Test Pass Fail. This variable contains the serial ids for the combination Wafer Number, Bin Number, Bin Pass Fail, Cond Pass Fail and X & Y Co-ordinates
calculateStandardDeviation This method is to calculate the standard deviation for the given test results
findMedian This method is to calculate the median for the given test data set
getFPseudoSigma This method is used to calculate the F-Pseudo Sigma based on the formula while is quartile https://bqs.usgs.gov/srs/SRS_Fall03/statrate.htm}
getPercentile This method is to calculate the given percentile value
getQuartile This method is to calculate the given Quartile(1 or 3)
getTestDataListForDataSummary This method is to get the actual test data to be considered to calculate the data summary It will filter out the duplicates and get the unique data
getTriMean This method is used to calculate the F-Pseudo Sigma based on the formula while is quartile https://www.statisticshowto.com/trimean}
getValueBasedOnScale This method is used to get the value based on the scale of the test, which was calculated based on test units.
