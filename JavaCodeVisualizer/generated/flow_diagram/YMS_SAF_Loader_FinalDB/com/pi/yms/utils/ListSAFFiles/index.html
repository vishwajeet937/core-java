com.pi.yms.utils.ListSAFFiles

This class is used to list out the valid folders and the files from them based on the configuration file called dc_auto_insert.cfg

No. of Methods With Description Without Description
3 3 0
Method Description
getErrorMessage This method is to return the error message for the error occurred while getting the list of SAF files
getSAFFilesBasedOnConfiguration To get the folders and SAF files from each folder
based on a given configuration file. (Ex: dc_auto_insert.cfg)
filesList will be passed to get the configuration of a specific files, which will be called from the loader
If searchInsertedFolder is passed as true, then it will search for the file in Inserted folder also(which are already loaded)
This will happen in case if the re-test file comes, then it need the already loaded files
If that file is loaded by Syntricity and no data available in Staging table, then it must be found in Inserted file to parse and read the test data
getValidFilesListFromFolder To get a list of files from a folder based on the valid list of extensions(Ex: .saf, .txt)
