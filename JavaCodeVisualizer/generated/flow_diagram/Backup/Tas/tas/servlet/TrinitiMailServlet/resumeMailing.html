Class-Name : TrinitiMailServlet

Method-Name : resumeMailing

Resumes resending mails if any in mail queue, which will be invoked when there is mail send request or servlet initialize phase. And mails will be tries to resend max 5 times only, if it fails still it will not entertain any more and removes from queue as well as deletes its backup xml file.
