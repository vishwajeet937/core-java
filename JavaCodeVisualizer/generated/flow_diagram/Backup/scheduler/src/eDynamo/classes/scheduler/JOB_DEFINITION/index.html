src.eDynamo.classes.scheduler.JOB_DEFINITION

Once you have the implementation of your job ,you have to define your job and this is one time creation.Here we will see how to define ECHO JOB defined above. 

JOB_DEFINITION jobDefinition = new JOB_DEFINITION();

jobDefinition.setJOB_TYPE_ID("007"); //ID SHOULD BE UNIQUE
jobDefinition.setJOB_TYPE("ECHO JOB");
jobDefinition.setJOB_CLASS("com.triniti...EchoJobImpl"); // Fully qualified class name of your job implmentation

JOB_PARAMS[] job_params = new JOB_PARAMS[2];

job_params[0] = new JOB_PARAMS();
job_params[0].setJOB_TYPE_ID("007");
job_params[0].setPARAM_ID("ECHO_P_ID1"); // PARAM_ID should be unique
job_params[0].setPARAMETER("echoString1"); // This parameter value should match keys used in the job implementation above

job_params[1] = new JOB_PARAMS();
job_params[1].setJOB_TYPE_ID("007");
job_params[1].setPARAM_ID("ECHO_P_ID2");
job_params[1].setPARAMETER("echoString2");

jobDefinition.setJOB_PARAMS(job_params);
