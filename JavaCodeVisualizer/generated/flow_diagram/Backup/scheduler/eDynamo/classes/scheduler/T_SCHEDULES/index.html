eDynamo.classes.scheduler.T_SCHEDULES

Creating a Cron Schedule

Schedule has to be created to use it for scheduling purposes.Before adding a schedule the details regarding the schedule are to be set.Following is the pseudo code for the creation of a cron schedule that has to run for every minute on all the week days ie the cron expression [* * * * 2-5] . 

...
T_SCHEDULES cronSchedule = new T_SCHEDULES();

cronSchedule.setSCHEDULE_TYPE("Cron Schedule");
cronSchedule.setSCHEDULE_TYPE_ID("001012");
cronSchedule.setDESCRIPTION("Schedule that runs for every minute on week days");
cronSchedule.setENABLED("true");
cronSchedule.setSCHEDULE_ID("0001111");
cronSchedule.setSCHEDULE_NAME("Every_Minute_Schedule");

Calendar calendar = Calendar.getInstance();
calendar.set(Calendar.MILLISECOND, 0);
calendar.set(2005, 8, 30, 18, 06, 0);   // Start date set to  18:06 August,30 2005
cronSchedule.setSTART_DATE(new Timestamp(calendar.getTimeInMillis()));

calendar.set(2006, 9, 30, 18, 06, 0);   //End date set to 18:06 September,30 2006
cronSchedule.setEND_DATE(new Timestamp(calendar.getTimeInMillis()));


//Creating the parameters for the cron schedule

T_SCHEDULE_PARAMS parameters_minutes = new T_SCHEDULE_PARAMS();
parameters_minutes.setPARAM_ID("001020");      //predefined
parameters_minutes.setPARAMETER("MINUTES");    //predefined
parameters_minutes.setPARAM_VALUE("*");
parameters_minutes.setSCHEDULE_ID("0001111"); //Set the Schedule Id from creating T_SCHEDULES

T_SCHEDULE_PARAMS parameters_hours = new T_SCHEDULE_PARAMS();
parameters_hours.setPARAM_ID("001022");         //predefined
parameters_hours.setPARAMETER("HOURS");         //predefined
parameters_hours.setPARAM_VALUE("*");
parameters_hours.setSCHEDULE_ID("00001111"); //Set the Schedule Id from creating T_SCHEDULES

T_SCHEDULE_PARAMS parameters_DOM = new T_SCHEDULE_PARAMS();
parameters_DOM.setPARAM_ID("001024");            //predefined
parameters_DOM.setPARAMETER("DATE");             //predefined
parameters_DOM.setPARAM_VALUE("*");
parameters_DOM.setSCHEDULE_ID("00001111");      //Set the Schedule Id from creating T_SCHEDULES

T_SCHEDULE_PARAMS parameters_month = new T_SCHEDULE_PARAMS();
parameters_month.setPARAM_ID("001026");         //predefined
parameters_month.setPARAMETER("MONTH");         //predefined
parameters_month.setPARAM_VALUE("*");
parameters_month.setSCHEDULE_ID("00001111");    //Set the Schedule Id from creating T_SCHEDULES

T_SCHEDULE_PARAMS parameters_DOW = new T_SCHEDULE_PARAMS();
parameters_DOW.setPARAM_ID("10");               //predefined
parameters_DOW.setPARAMETER("TIME");            //predefined
parameters_DOW.setPARAM_VALUE("2-5");
parameters_DOW.setSCHEDULE_ID("00001111");      //Set the Schedule Id from creating T_SCHEDULES

T_SCHEDULE_PARAMS[] params_array  = new T_SCHEDULE_PARAMS[5];
params_array[0] = parameters_minutes;
params_array[1] = parameters_hours;
params_array[2] = parameters_DOM;
params_array[3] = parameters_month;
params_array[4] = parameters_DOW;

cronSchedule.setT_SCHEDULE_PARAMS(params_array);  //Set the parameters to the schedule

After creating the Schedule object it can be added to the databse by calling the SchedulerAPI method as follows 

...
SchedulerAPI schedulerAPI = new SchedulerAPI();
schedulerAPI.addSchedule(cronSchedule);
...
