Package : com.pi.yms.utils

Class : ListSAFFiles

Method : getSAFFilesBasedOnConfiguration

To get the folders and SAF files from each folder
based on a given configuration file. (Ex: dc_auto_insert.cfg)
filesList will be passed to get the configuration of a specific files, which will be called from the loader
If searchInsertedFolder is passed as true, then it will search for the file in Inserted folder also(which are already loaded)
This will happen in case if the re-test file comes, then it need the already loaded files
If that file is loaded by Syntricity and no data available in Staging table, then it must be found in Inserted file to parse and read the test data
